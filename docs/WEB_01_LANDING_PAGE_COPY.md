## 4) Section "Installation rapide"

### Linux

Telecharger le paquet `.deb`, installer, puis verifier avec `algolab --help`.

### macOS

Telecharger le binaire, `chmod +x`, puis lancer `./algolab --help`.

### Windows

Telecharger `algolab.exe`, ouvrir PowerShell et executer `.\algolab.exe --help`.

### Extension VS Code

Installer le fichier `.vsix` depuis "Install from VSIX..." dans VS Code.
