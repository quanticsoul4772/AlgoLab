# Pack web/documentation AlgoLab

Ce dossier contient un pack pret pour construire :

- la landing page,
- la documentation utilisateur,
- la FAQ,
- les pages de release/changelog.

## Fichiers a utiliser

- `WEB_00_PLAN_GLOBAL.md` : vision, parcours et checklist globale.
- `WEB_01_LANDING_PAGE_COPY.md` : contenu redactionnel de la landing page.
- `WEB_02_DOCUMENTATION_STRUCTURE.md` : architecture de la documentation.
- `WEB_03_GETTING_STARTED.md` : page "demarrage rapide".
- `WEB_04_INSTALLATION_MULTI_OS.md` : installation Linux/macOS/Windows.
- `WEB_05_PAGE_EXTENSION_VSCODE.md` : page extension VS Code.
- `WEB_06_FAQ_TROUBLESHOOTING.md` : FAQ et depannage.
- `WEB_07_MODELES_PAGES.md` : templates reutilisables.

## Recommandation d'usage

1. Commencer par `WEB_00_PLAN_GLOBAL.md`.
2. Integrer `WEB_01_LANDING_PAGE_COPY.md` dans la home.
3. Construire la navigation de doc avec `WEB_02_DOCUMENTATION_STRUCTURE.md`.
4. Publier les pages de base : installation, getting started, FAQ.
