# AlgoLab

Interpreteur pedagogique d un pseudo code, base sur Lark. Le projet couvre la grammaire, l AST, l execution et un systeme d erreurs pedagogoques.

## Demarrage rapide
- Documentation: voir docs/
- Code: src/algolab/
- Exemples: examples/
- Tests: tests/

## Execution simplifiee
- Lancer un fichier: scripts/run.sh examples/hello_world.algo
- Lancer du code inline: scripts/run.sh -c 'Variable x : Entier Debut x <- 3 Ecrire x Fin'
