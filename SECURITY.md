# Security Policy

| Version | Supportée |

## Signaler une vulnérabilité

Si vous découvrez une vulnérabilité de sécurité dans AlgoLab, merci de **ne pas** ouvrir une issue publique.

Envoyez plutôt un email à : **igoradande44@gmail.com**

Vous recevrez une confirmation sous 48h. Nous nous engageons à :

1. Confirmer la réception de votre rapport
2. Évaluer la vulnérabilité
3. Publier un correctif dans les meilleurs délais
4. Vous créditer dans le changelog (sauf si vous préférez rester anonyme)

Merci de contribuer à la sécurité d'AlgoLab !
